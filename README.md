# Magic Mirror – Hand Tracking

### Currently, the project only implements the **gesture detection and hand tracking system**. UI widgets and full mirror functionality are planned for future updates.

## Features
- Real-time hand tracking via camera
- Gesture-based interaction
- Designed for a modular widget system (planned)

## Planned Widgets
- Weather
- Music
- Clock
- News
- ...

## Tech Stack
- React
- JavaScript
- Hand Tracking Library (e.g. MediaPipe)

## Status
⚠️ Early development — only gesture recognition is implemented.
